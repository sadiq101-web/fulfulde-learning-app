# Fulfulde Learning App

A simple web application designed to help people learn the **Fulfulde (Fulani) language**.

This project aims to support **language preservation** and **digital literacy** for Fulani speakers and learners across West Africa.

---

## Features

- Learn useful Fulfulde phrases  
- Audio pronunciation  
- Quiz mode  
- Simple mobile-friendly design  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## Purpose

Fulfulde is widely spoken across West Africa but has **limited digital learning tools**.  
This project was created to promote **Fulfulde language learning** and **cultural preservation** through technology.

---

## Getting Started

1. Download or clone the project repository.  
2. Open the project folder.  
3. Open `index.html` in a web browser.  
4. Start learning Fulfulde!
